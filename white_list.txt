# 系统广播
# 无法接受系统广播，会导致来电提醒类APP无法随电话启动
# 此类APP通常都应当添加到白名单中
#
org.xdty.callerinfo
me.gitai.smscodehelper


# 消息推送
# 可能无法及时收到推送，但打开APP时仍会收到消息
# 如果只是偶尔打开查下消息，则不需要添加到白名单中
# 这会导致滴答清单提醒功能失效
#
com.tencent.mm
# com.tencent.mobileqq
# com.microsoft.office.outlook
# com.tencent.androidqqmail
cn.ticktick.task


# 系统监控
# 此类APP由辅助功能(Accessibility)保证运行，不需要添加白名单
#
# com.oasisfeng.greenify
# com.coolapk.autoinstaller
